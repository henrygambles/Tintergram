# TinterGram	

 (or T'interGram)	

 ![tintergram gif](https://media.giphy.com/media/l1J3OTuefx2MopjFe/giphy.gif)	

 ## Quickstart	

 First, clone this repository. Then:	

 ```bash	
> bundle	
> rake db:create	
> rake db:migrate	
> rspec # Run the tests to ensure it works	
> rails s # Start the server at localhost:3000	
```
